# Changelog

## 3.1.0 - 2026-04-08

### Features
- Added a Changelog page with a navigation link so the project history can be viewed directly from the app.
- Added per-category padlock controls on the home page to lock or unlock drag-and-drop project reordering.
- Added drag-and-drop sorting for projects within each category, matching the lock/unlock workflow used in RewardLedger settings pages.

### Improvements
- Added persistent `sort_order` support so project order stays saved after refresh.
- Updated the footer to read the app version from the `VERSION` file.
